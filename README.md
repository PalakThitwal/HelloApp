# HelloApp

HelloApp is a Java console application that demonstrates step-by-step programming concepts.  
It starts with Hello World and gradually adds user input, collections, file handling, and object-oriented design.

## Features

- Display Hello World
- Accept name from command-line argument
- Handle multiple command-line names
- Accept name from keyboard input
- Store names in an ArrayList
- Remove a name
- Refactor into methods
- Use a separate NameManager class
- Save names to a file (names.txt)
- Display greetings in banner format

## How to Run

Compile:

javac HelloApp.java NameManager.java

Run:

java HelloApp
